🧰Tech Stack

🖥️ Frontend

  • React 19 + TypeScript + Vite
  • React Router 7 — SPA routing
  • Chart.js — กราฟรายงาน
  • FullCalendar — ตารางนัดหมาย
  • Vitest + Testing Library
  • สถาปัตยกรรมแบบ feature-based + multi-tenant

⚙️ Backend

  • Go 1.25 + Gin framework
  • PostgreSQL 16 ผ่าน pgx v5
  • sqlc — type-safe SQL
  • golang-migrate — จัดการ schema
  • JWT authentication
  • รันด้วย Docker Compose
